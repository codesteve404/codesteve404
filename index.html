Introduction

Dark Souls 2: Scholar of the First Sin is the remastered version of Dark Souls 2, with all the Downloadable Contents (DLC) added into it. It's a Japanese action role-playing game (JRPG) developed by FromSoftware and published by Bandai Namco Games. It is notorious for punishing you if you try to rush head-first into certain areas of the game. The remaster changes where the monsters are located, adds new items and changes the place of the old items. This can make it difficult to complete for those who played the base game before, since they need to relearn where the items are. It also adds two new rings which can be acquired by completing two challenges. The Illusory Ring of a Conqueror hides the item in your character's left hand and can be obtained by clearing the game without the character dying. The Illusory Ring of the Exalted hides the item in your character's right hand and can be obtained by clearing the game without using a single bonfire, which are equivalent to checkpoints in other games. This is a guide for the no-death run, which is the easier of the two. The expression no-death run will be used throughout the website, run simply meaning completing the game.
